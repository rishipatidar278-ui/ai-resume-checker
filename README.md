# AI Resume Checker Bot (HTML, CSS, JavaScript)

A lightweight, client-side “AI-style” resume checker that analyzes pasted resume text and generates an ATS-like score, detects skills, and provides improvement suggestions. Runs fully in the browser (no backend required).

## Features
- ATS-style score (0–100) with animated progress bar
- Skill detection using a configurable keyword list
- Section checks (Experience, Education, Projects, Certifications)
- Actionable suggestions based on missing sections and low score
- Works offline after first load (static files)

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
